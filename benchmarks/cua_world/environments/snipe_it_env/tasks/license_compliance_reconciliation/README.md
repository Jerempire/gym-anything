# Task: license_compliance_reconciliation

## Occupation Context
**Role:** Software Asset Manager
**Industry:** Technology

## Task Description
Reconcile three existing software licenses based on annual audit findings and create one new license. Agent must update seat counts, costs, expiration dates, and order numbers across three different licenses, then create a completely new Slack license entry.

## Difficulty: very_hard
- Must navigate to Licenses section (not the default view)
- Must find and edit 3 separate licenses with different field changes
- Must create a new license with multiple required fields
- Must not modify any other licenses
- No UI path provided
