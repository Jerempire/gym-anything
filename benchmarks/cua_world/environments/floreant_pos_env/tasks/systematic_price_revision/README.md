# Task: Systematic Price Revision with Premium Tier Launch

## Overview
**Role**: Restaurant Manager / Owner
**Difficulty**: Very Hard
**Domain**: Restaurant POS Menu Pricing Management

## Business Context
A quarterly food cost and competitive analysis revealed that the restaurant is underpriced on several items and is missing a premium tier to attract higher-value customers. The manager must update prices on key existing items and launch a new "PREMIUM SELECTIONS" category with upscale menu items.

## Task Requirements

### Part 1: Price Updates on Existing Items
Find and update these three existing items in the POS system:

| Item | Old Price | New Price |
|------|-----------|-----------|
| HAMMER COFFEE | $2.00 | **$3.75** |
| SMK HOUS B FAST | (current) | **$10.50** |
| OLD TIMER B FAST | (current) | **$9.75** |

### Part 2: New Premium Category and Items
Create a new menu category called **PREMIUM SELECTIONS** and add these three items:

| Item | Price |
|------|-------|
| WAGYU BEEF BURGER | $24.95 |
| TRUFFLE FRIES | $12.95 |
| ARTISAN CHEESE PLATE | $18.50 |

## Why This Is Hard
- Must find items among 100+ items in the database (search or scroll)
- Each price update requires: navigate to item → Edit → Secret Key dialog → change price → Save
- Must remember exact new prices for each item
- Must also create new category and 3 items (context switching)
- 3 updates + 1 category + 3 new items = 7 separate UI interactions

## Scoring (100 points)
- HAMMER COFFEE updated to $3.75: 15 pts
- SMK HOUS B FAST updated to $10.50: 15 pts
- OLD TIMER B FAST updated to $9.75: 15 pts
- PREMIUM SELECTIONS category created: 15 pts
- WAGYU BEEF BURGER at $24.95: 10 pts
- TRUFFLE FRIES at $12.95: 10 pts
- ARTISAN CHEESE PLATE at $18.50: 10 pts
- All items correct (bonus): 10 pts
- Pass threshold: 60 points
