# Task: Patio Section with Cocktail Menu

## Overview
**Role**: Restaurant Manager
**Difficulty**: Very Hard
**Domain**: Restaurant POS Floor Plan & Menu Configuration

## Business Context
A restaurant is expanding its service area to include a new outdoor patio for the summer season. The POS system must be configured to handle the new floor section (so servers can assign orders to patio tables) as well as a new patio-specific menu with specialty cocktails and food items.

## Task Requirements

### Part 1: Create Patio Floor Section
Create a new restaurant floor section called **PATIO** and add six tables to it:
- P1, P2, P3, P4, P5, P6

### Part 2: Create Liquor Tax
Create a new tax rate: **LIQUOR TAX** at **9.5%**

### Part 3: Create Patio Specials Menu
Create a new menu category **PATIO SPECIALS** with five items:

| Item | Price |
|------|-------|
| SANGRIA PITCHER | $18.00 |
| PALOMA COCKTAIL | $11.00 |
| DRAFT BEER PATIO | $6.50 |
| LOADED NACHOS | $13.95 |
| GRILLED CORN | $4.75 |

## Why This Is Hard
This task requires navigating four completely different areas of the Back Office:
1. Floor plan editor (to create the section and tables)
2. Tax configuration
3. Menu category creation
4. Menu item creation (5 items)

Each area has its own UI flow, and the floor plan editor is particularly complex as it requires creating both the floor section AND individual tables within it.

## Scoring (100 points)
- PATIO floor section created: 15 pts
- Each of 6 tables in PATIO (4 pts each): 24 pts
- LIQUOR TAX at 9.5%: 15 pts
- PATIO SPECIALS category: 10 pts
- Each of 5 items with correct name (4 pts each): 20 pts
- Each of 5 items with correct price (3.2 pts each): 16 pts
- Pass threshold: 60 points
