# Create Chart Task

**Difficulty**: 🟡 Medium
**Estimated Steps**: 40
**Timeout**: 300 seconds (5 minutes)

## Objective

Create a bar or column chart from provided sales data. This task tests chart creation, data visualization, and understanding of chart wizards.

## Starting State

- LibreOffice Calc opens with monthly sales data
- Data contains: Month and Sales columns
- 6 months of data (January through June)

## Data Layout

| Month    | Sales  |
|----------|--------|
| January  | 12500  |
| February | 15200  |
| March    | 14800  |
| April    | 16300  |
| May      | 17900  |
| June     | 18500  |

## Required Actions

1. Select the data range (A1:B7 or similar)
2. Navigate to Insert → Chart menu
3. Choose chart type: Bar or Column chart
4. Configure the chart data range
5. Add chart title (optional but recommended)
6. Position the chart in the spreadsheet
7. Save the file as ODS format

## Success Criteria

1. ✅ Chart object exists in the spreadsheet
2. ✅ Chart type is bar or column (not line, pie, etc.)
3. ✅ Chart has data range configured (points to sales data)
4. ✅ Sales data preserved in spreadsheet

**Pass Threshold**: 75% (3 out of 4 criteria)

## Skills Tested

- Data range selection
- Chart wizard navigation
- Chart type selection
- Visual data representation
- Object positioning in spreadsheets

## Tips

- Select data including headers before inserting chart
- Use Insert → Chart (or Insert → Object → Chart)
- Column chart shows vertical bars, Bar chart shows horizontal bars
- Either is acceptable for this task
- The chart wizard guides you through multiple steps
- You can move the chart by dragging it after creation
