# Science Fair Judge Score Sheet Creator Task

**Difficulty**: 🟡 Medium  
**Skills**: Form design, cell merging, borders, text formatting, layout planning  
**Duration**: 180 seconds (3 minutes)  
**Steps**: ~15

## Objective

Create a professional, printable judge score sheet for a school science fair using LibreOffice Calc as a form design tool. This task tests formatting skills, layout design, and understanding of how to create human-usable paper forms.

## Scenario Context

Maria is a parent volunteer at Hillside Elementary School. The 4th-5th grade science fair is THIS Friday (3 days away). The organizing committee just realized they never created judge score sheets—each project needs to be evaluated by 3 judges across multiple criteria. Maria needs to create something functional and professional-looking in 30 minutes using only LibreOffice Calc on her laptop.

## Task Description

The agent must create a printable score sheet that includes:

### Required Elements:
1. **Header Section**: Title identifying the form (e.g., "Science Fair Judge Score Sheet")
2. **Participant Information**: Fields for Student Name and Project Number
3. **Four Scoring Categories** (each worth 0-25 points):
   - Creativity
   - Scientific Method
   - Presentation Quality
   - Clarity of Explanation
4. **Comment Spaces**: Area for brief comments under each category
5. **Total Score**: Field showing total out of 100 points
6. **Judge Information**: Fields for Judge Name, Date, and Signature

### Design Requirements:
- Professional appearance (will be used by district administrators)
- Adequate space for handwritten entries
- Clear visual hierarchy (headers, sections, fields)
- Appropriate use of borders to define form structure
- Must fit on one printed page (8.5" x 11")
- 60 copies needed, so keep design print-friendly

## Starting State

- LibreOffice Calc opens with a blank spreadsheet
- Requirements document available on Desktop for reference
- Agent must design the form from scratch

## Expected Actions

1. Plan form layout (header, sections, fields)
2. Create and format header section with merged cells
3. Add participant information fields with appropriate spacing
4. Build scoring criteria table with:
   - Category names (bold, clear)
   - Point ranges (0-25)
   - Score entry fields
   - Comment areas with adequate space
5. Create prominent total score section (out of 100)
6. Add judge information footer fields
7. Apply consistent borders throughout
8. Adjust row heights for adequate writing space
9. Verify print layout (fits on one page)
10. Save as science_fair_scoresheet.ods

## Success Criteria

1. ✅ **All required fields present** (10+ distinct elements: header, student info, 4 categories, totals, judge info)
2. ✅ **Proper formatting applied** (merged cells for headers, borders for structure, bold emphasis)
3. ✅ **Logical layout structure** (clear sections, visual hierarchy, proper spacing)
4. ✅ **Print-ready dimensions** (content fits within reasonable page bounds)
5. ✅ **Adequate writing space** (sufficient row heights for handwritten entries)
6. ✅ **Calculation correctness** (if total formula used, sums to 100 points)
7. ✅ **Professional appearance** (consistent styling, clean presentation)

**Pass Threshold**: 70% (requires at least 5 out of 7 criteria)

## Skills Tested

- **Cell Merging**: Combine cells for headers and form fields
- **Border Application**: Create form structure with varied border styles
- **Text Formatting**: Use bold, font sizes, alignment for hierarchy
- **Row/Column Sizing**: Adjust dimensions for optimal spacing
- **Layout Design**: Create usable, professional-looking forms
- **Print Planning**: Consider how digital design translates to paper
- **Visual Hierarchy**: Emphasize important elements appropriately

## Verification Strategy

The verifier examines the spreadsheet to ensure:
- All required text elements are present (categories, fields, labels)
- Formatting indicates professional design effort (borders, emphasis)
- Layout is logical and fits within printable dimensions
- If formulas are used, they calculate correctly (4 categories × 25 = 100)
- Form would be usable for manual data collection

## Tips

- Use cell merging for headers: Select cells → Format → Merge Cells
- Apply borders: Format → Cells → Borders (or use toolbar button)
- Make text bold: Ctrl+B or Format → Text → Bold
- Adjust row height: Right-click row number → Row Height
- Center text: Use alignment buttons in toolbar
- Preview print layout: File → Print Preview
- Remember this is for HANDWRITING - leave adequate space!

## Common Pitfalls

- Forgetting to merge cells for header areas
- Insufficient row height for handwritten comments
- Missing borders to define form structure
- Cramming too much content (won't fit on one page)
- Not making category names bold/prominent
- Forgetting signature or date fields
- Using font sizes too small for printed form
