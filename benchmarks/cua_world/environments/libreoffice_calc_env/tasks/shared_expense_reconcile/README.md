# Shared Expense Reconciliation Task

**Difficulty**: 🟡 Medium  
**Skills**: Formula creation, financial calculation, conditional formatting, multi-person accounting  
**Duration**: 180 seconds  
**Steps**: ~15

## Objective

Reconcile shared expenses between three roommates (Alice, Bob, and Carol) by calculating individual totals, determining net balances, and creating a settlement summary. This task tests formula creation, financial reasoning, and practical accounting skills commonly used by roommates, travel groups, and friends splitting costs.

## Task Description

The agent must:
1. Work with a pre-populated expense log showing one month of shared expenses
2. Calculate how much each person paid in total (using SUMIF formulas)
3. Calculate how much each person owes (based on equal splits and assigned expenses)
4. Determine net balances (Paid - Owed)
5. Verify mathematical consistency (all balances sum to zero)
6. Apply conditional formatting to highlight debts and credits
7. Create settlement instructions

## Pre-populated Data

**Expense Log (Rows 1-11):**
- Date, Description, Amount, Paid By, Split Type columns
- 10 expenses including groceries, utilities, rent, personal items
- Some expenses split equally among 3 people
- Some expenses assigned to specific individuals
