# Weighted Grade Calculator Task

**Difficulty**: 🟡 Medium  
**Skills**: Weighted averages, percentage calculations, nested IF statements  
**Duration**: 180 seconds  
**Steps**: ~50

## Objective

Calculate final course grades using a weighted grading scheme and assign letter grades based on standard grading scales. This task tests understanding of weighted averages, percentage operations, and multi-condition IF statements.

## Task Description

The agent must:
1. Open a LibreOffice Calc spreadsheet with student assessment data (Homework, Midterm, Final Exam)
2. Create formulas to calculate weighted final grades using:
   - Homework: 30% of final grade
   - Midterm: 30% of final grade
   - Final Exam: 40% of final grade
3. Create nested IF formulas to assign letter grades:
   - A: ≥90%
   - B: 80-89%
   - C: 70-79%
   - D: 60-69%
   - F: <60%
4. Apply formulas to all students
5. Save the file

## Grading Scheme

**Weighted Components:**
- Homework (Column B): 30%
- Midterm (Column C): 30%
- Final Exam (Column D): 40%

**Letter Grade Scale:**
- A: 90-100%
- B: 80-89%
- C: 70-79%
- D: 60-69%
- F: 0-59%
