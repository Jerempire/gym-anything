# Paint Inventory Calculator for Home Renovation Task

**Difficulty**: 🟡 Medium  
**Skills**: Formulas, area calculation, ROUNDUP function, cost estimation  
**Duration**: 240 seconds  
**Steps**: ~30

## Scenario

You're helping a frustrated homeowner who's midway through painting their house. They have scattered measurements, receipts from different stores, and need to figure out EXACTLY how much paint to buy before the hardware store closes. The data is messy - some ceiling heights are missing, paint prices are in notes, and they need calculations FAST.

## Objective

Organize room dimensions and calculate:
1. **Wall surface area** for each room (perimeter × height - door/window areas)
2. **Paint quantity needed** in whole gallons (can't buy 2.3 gallons!)
3. **Total cost** including 7% sales tax

## Starting Data

CSV file with partial information:
- Room names and dimensions (some incomplete)
- Door and window counts  
- Paint color notes
- Scattered price information

**Missing/Messy Elements:**
- Some ceiling heights are blank (should default to 8 ft)
- No formulas - just raw measurements
- Calculations need to be created
- Prices mentioned in notes, not organized columns
