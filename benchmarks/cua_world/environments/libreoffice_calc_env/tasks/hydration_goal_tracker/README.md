# Daily Hydration Goal Tracker Task

**Difficulty**: 🟡 Medium  
**Skills**: Formulas (SUM, AVERAGE, IF, COUNTIF), cell references, conditional logic  
**Duration**: 180 seconds (3 minutes)  
**Steps**: ~15

## Objective

Analyze daily water intake data to determine if hydration goals are being met. Add formulas to calculate daily totals, compare against a 64 oz goal, compute statistics, and identify patterns. This task tests practical spreadsheet skills for personal health tracking.

## Task Description

You have tracked your water intake for 14 days, recording how much you drank in the morning, afternoon, and evening (in ounces). The health recommendation is 64 oz per day. Now you need to:

1. Calculate the daily total water intake
2. Determine if each day met the 64 oz goal
3. Calculate average daily intake over the 2-week period
4. Count how many days you successfully met your goal

## Starting Data

The spreadsheet contains:
- **Column A**: Date (14 days)
- **Column B**: Morning Intake (oz)
- **Column C**: Afternoon Intake (oz)
- **Column D**: Evening Intake (oz)
- **Columns E-F**: Empty (you will add formulas here)
