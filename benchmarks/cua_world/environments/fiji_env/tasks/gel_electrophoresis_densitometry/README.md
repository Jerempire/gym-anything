# Gel Electrophoresis Densitometry

## Task Overview

**Occupation**: Biochemist and Biophysicist / Molecular and Cellular Biologist
**Difficulty**: Very Hard
**Timeout**: 720 seconds / 75 max steps

A biochemist has run an SDS-PAGE protein gel to compare expression levels across multiple experimental conditions (lanes). The task is to quantify band intensities using Fiji's built-in gel analysis tools, apply background correction, and express all lanes relative to a reference loading control.

## Dataset

**Source**: Real SDS-PAGE gel image from the ImageJ/Fiji sample library (`gel.gif`)
- Downloaded from: `https://imagej.nih.gov/ij/images/gel.gif`
- Converted to TIFF: `~/Fiji_Data/raw/gel/protein_gel.tif`
- Image type: 8-bit grayscale, Coomassie blue-stained polyacrylamide gel
- Content: Multiple protein samples separated by molecular weight across several lanes
- Darker areas indicate higher protein concentration (conventional staining convention)

The gel.gif image is a classic real laboratory gel that has been part of the ImageJ sample dataset for decades and is widely used for teaching gel analysis.

## Biological Context

Gel densitometry is a fundamental technique in molecular biology for:
- Quantifying protein expression levels across different samples or conditions
- Comparing relative abundance of specific proteins between lanes
- Normalizing experimental samples to a loading control lane
- Validating Western blot or Coomassie-stained gel results

In this scenario:
- Lane 1 is the **reference/loading control** lane (normalized intensity = 1.0)
- Lanes 2+ are experimental samples whose expression is measured relative to Lane 1
- Background correction removes baseline noise within each lane profile

## Fiji Gel Analysis Workflow

Fiji (ImageJ) provides a dedicated gel analysis workflow under `Analyze > Gels`:

1. **Open the gel image**: `File > Open` then navigate to `~/Fiji_Data/raw/gel/protein_gel.tif`
2. **Check orientation**: Darker bands = more protein. If bands appear bright on a dark background, invert: `Edit > Invert` (Ctrl+Shift+I)
3. **Select First Lane**: Draw a rectangular selection around Lane 1, then `Analyze > Gels > Select First Lane` (or press `1`)
4. **Select Next Lane**: Move rectangle to Lane 2, then `Analyze > Gels > Select Next Lane` (or press `2`). Repeat for all lanes.
5. **Plot Lanes**: `Analyze > Gels > Plot Lanes` (or press `3`) — opens profile plots for each lane
6. **Measure peaks**: In the plot window, use `Analyze > Gels > Label Peaks` to compute % area for each peak
7. **Record intensities**: The Results table contains peak areas (% total) that represent band intensities
8. **Apply background subtraction**: Use the "straight line" tool in the profile plot to draw baselines for rolling ball correction per lane

## Tips for the Agent

- The gel image has lanes oriented vertically (proteins run from top to bottom). Lane selections should be vertical rectangles of equal width.
- If the Analyze > Gels menu is grayed out, make sure the image is the active window and you have drawn a selection first.
- Fiji's gel tool assigns sequential lane numbers internally; the first selection becomes Lane 1.
- For normalized_intensity: divide each lane's raw_intensity by Lane 1's raw_intensity. Lane 1 itself should have normalized_intensity = 1.0.
- If gel.gif was not downloaded, try: `File > Open Samples > Gel` in Fiji to load the built-in sample gel.
- The gel_info.txt file at `~/Fiji_Data/raw/gel/gel_info.txt` contains additional workflow guidance.
