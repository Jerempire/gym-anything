SportsFit Shop - Conversion Tracking Requirements
==================================================

Site in Matomo: SportsFit Shop

Please implement all of the following conversion goals for the SportsFit Shop
website in Matomo. The goals represent the stages of a typical e-commerce
purchase funnel.

--------------------------------------------------------------------
Goal 1: Product Page View
  Purpose: Track when a visitor views any product detail page
  Match method: Visit URL contains "/products/"
  Revenue tracking: None (leave at 0)

--------------------------------------------------------------------
Goal 2: Add to Cart
  Purpose: Track when a visitor adds an item to their cart
  Match method: Visit URL contains "/cart/add"
  Revenue tracking: None (leave at 0)

--------------------------------------------------------------------
Goal 3: Checkout Started
  Purpose: Track when a visitor begins the checkout process
  Match method: Visit URL contains "/checkout"
  Revenue tracking: None (leave at 0)

--------------------------------------------------------------------
Goal 4: Purchase Confirmation
  Purpose: Track completed purchases — the primary revenue conversion
  Match method: Visit URL matches EXACTLY "/order/thank-you"
           (use exact URL match, not "contains")
  Revenue tracking: Set to 0 (revenue tracked separately via ecommerce module)

--------------------------------------------------------------------

Notes:
- All goals should be created for the SportsFit Shop site
- Use "Visit URL" as the trigger type for all goals
- Goal names must match exactly as listed above
- For Goal 4, use exact match (not "contains") to avoid false positives
  from pages like /order/thank-you-for-subscribing

--------------------------------------------------------------------
End of specification
