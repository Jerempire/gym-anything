# crew_partner_network_setup

**Difficulty:** Hard  
**Environment:** Flight Crew View (FLICA), Android AVD API 34

## Task Description

A United Airlines Captain based at Chicago O'Hare (ORD) sets up their Flight Crew View profile.

## What the Agent Must Do

1. Change display name to "Capt. Rodriguez | ORD" (CUA Suite setting)
2. Set position to "Captain" (Position setting)
3. Set home airport to ORD and base/activity airport to ORD (HOME & BASE AIRPORTS setting)
4. Add three crew colleagues as friends: fo.james@united.aero, fa.chen@united.aero, fa.garcia@united.aero

## Features Used

- CUA Suite (display name text input)
- Position dropdown (Captain)
- HOME & BASE AIRPORTS section (two airport code dropdowns)
- Friends (add contacts)
