# Legal Contract Redline

## Task Description
Clean up a messy draft Vendor Services Agreement to meet professional legal document standards.

## Occupation Context
Paralegal at a corporate law firm preparing documents for partner review.

## Difficulty: Very Hard

## Challenges
- Discover inconsistent heading hierarchy across 8 sections
- Find and fix inconsistent party name references (4+ variants)
- Complete definitions section with 3 missing defined terms
- Format deliverables table with professional styling
- Fix improperly centered body paragraphs
- Remove spurious italic/bold formatting from 3 sections
- Add signature block for both parties
- All issues must be discovered by the agent (not enumerated step-by-step)

## Data Source
Standard vendor services agreement structure based on ABA Model Agreements patterns.

## Verification
- Heading hierarchy consistency (python-docx style check)
- Party name consistency (text analysis)
- Definitions completeness (pattern matching)
- Table header formatting (bold + shading)
- Body alignment (no improper centering)
- Formatting cleanup (spurious styles removed)
- Signature block presence (keyword detection)
- VLM visual verification
