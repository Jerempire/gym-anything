# Task: broken_leo_mission_diagnosis

## Domain Context

**Primary occupation**: Atmospheric and Space Scientist (ONETSOC 19-2021.00)
**Workflow type**: Error diagnosis and correction in mission design scripts

Real astrodynamics engineers routinely audit colleague-prepared GMAT scripts before launch readiness reviews. A physically inconsistent orbital script would cause mission failure — incorrect inclination means the satellite never achieves sun-synchronous ground track, wrong SMA means the wrong altitude, inflated eccentricity risks re-entry at periapsis, and an implausible drag area corrupts lifetime predictions. Diagnosing and correcting all errors from first principles (without being told which parameters are wrong) is a core professional skill.

## Goal

Diagnose and correct all physical inconsistencies in the GMAT mission script at `/home/ga/Documents/missions/leo_comms_mission.script`, which was prepared for a **550 km altitude sun-synchronous dawn-dusk communications satellite** but contains several physically impossible or operationally wrong parameters.

After correcting all errors, propagate the orbit for 7 days and write a diagnosis report to `/home/ga/GMAT_output/leo_diagnosis_report.txt`.

## Success Criteria

The corrected script must be physically consistent with a 550 km sun-synchronous orbit:

## Injected Errors (Ground Truth)

The script contains exactly 4 injected physical errors:

1. `SMA = 7300.0` km — implies 929 km altitude, not 550 km
2. `ECC = 0.05` — implies a 350 km periapsis (nearly inside Earth's atmosphere at this SMA)
3. `INC = 55.0` deg — completely wrong for sun-synchronous; causes ~−6.7 deg/day RAAN drift instead of the required +0.9856 deg/day
4. `DragArea = 0.0001` m² — physically impossible for a 520 kg spacecraft (implies ~1 cm² cross-section)

## Orbital Mechanics Reference

**Sun-synchronous orbit condition** (J2 precession rate = Earth's mean motion around Sun):

```
cos(INC_ss) = -3/2 * (J2 * Re² * n) / (a² * n_sun)
```

For 550 km altitude (SMA = 6921.14 km): **INC_ss ≈ 97.73°**

RAAN drift for sun-sync: **+0.9856 deg/day** (tracks the Sun's apparent motion)
