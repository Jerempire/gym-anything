# retail_showroom_visual_merchandising

**Occupation**: Merchandise Displayer / Visual Merchandiser
**Industry**: Retail / Fashion
**Difficulty**: Extremely Hard

## Task Description

Design a flagship fashion retail showroom for Luxe & Co. The building shell is pre-drafted; the agent must use **4 distinct Sweet Home 3D features** to create an immersive retail experience:

1. **Furniture placement** -- furnish the showroom with display fixtures, VIP lounge seating, checkout counters, ambient lighting, and decorative greenery
2. **Room definition** -- define room boundaries for each merchandising zone so the floor plan communicates the layout
3. **Floor color** -- apply distinct floor colors or textures to each zone to visually differentiate merchandising areas and guide customer flow
4. **3D photo rendering** -- generate a 3D photo of the completed showroom and save it to the Desktop for the client presentation

## Scoring

| Criterion | Points | Requirement |
|-----------|--------|-------------|
| C1: Display fixtures | 25 | >=12 shelves + tables combined |
| C2: Room zones w/ floor color | 20 | >=3 rooms with floorColor/floorTexture set (partial >=1 -> 10) |
| C3: VIP + checkout | 15 | >=3 sofas + >=2 desks + >=2 chairs (partial scoring) |
| C4: Ambient decor | 25 | >=8 lamps + >=5 plants (partial >=4+2 -> 12) |
| C5: 3D photo + total + save | 15 | 3D photo exists (5) + >=45 total (5) + file changed (5) |
| **Total** | **100** | **Pass: 70** |

Wrong-target gate: <8 furniture items = score 0.
