# Sample Emails

This directory contains sample email files in mbox format that will be pre-populated into Thunderbird's Local Folders.

## Files

- **Inbox** - Sample incoming emails
- **Sent** - Sample sent emails
- **Drafts** - Sample draft emails

## Format

Emails should be in mbox format (the default for Thunderbird Local Folders).

Each email starts with a "From " line (note the space after From), followed by standard email headers and body.

Example:
```
From - Mon Jan 01 00:00:00 2024
From: sender@example.com
To: recipient@example.com
Subject: Test Email
Date: Mon, 1 Jan 2024 00:00:00 +0000
Message-ID: <unique-id@example.com>

Email body goes here.

```

## Customization

You can replace or add email files here to pre-populate different scenarios for testing.
