MORRISON & FOERSTER LLP
DOCUMENT FORMATTING STANDARDS GUIDE
Version 4.2 — Effective January 1, 2024

═══════════════════════════════════════════════════════════════════
SECTION 1: PARAGRAPH STYLE REQUIREMENTS
═══════════════════════════════════════════════════════════════════

All transactional documents (contracts, agreements, leases, amendments)
must adhere to the following paragraph style requirements when prepared
in Apache OpenOffice Writer or compatible word processors:

1.1 HEADING HIERARCHY

  Main Section Headings:
    - All primary numbered sections (e.g., "1. PARTIES", "2. PREMISES",
      "3. LEASE TERM", "10. DEFAULT AND REMEDIES") MUST use the
      "Heading 1" paragraph style.
    - Direct formatting (manually applied bold, font sizes) is NOT
      acceptable and violates document standards.

  Subsection Headings:
    - All secondary numbered headings (e.g., "1.1 Landlord",
      "4.2 Annual Rent Adjustments", "9.3 Mutual Indemnification")
      MUST use the "Heading 2" paragraph style.
    - Again, direct/manual formatting is strictly prohibited.

  Tertiary Headings (if present):
    - Any third-level headings (e.g., "1.1.1", "2.3.a") must use
      "Heading 3" paragraph style.

1.2 BODY TEXT

  All body paragraphs must use "Default Paragraph Style" or
  "Text Body" — never use manually typed spacing or font changes
  to distinguish paragraphs from headings.

SECTION 2: REQUIRED DOCUMENT ELEMENTS
═══════════════════════════════════════════════════════════════════

2.1 TABLE OF CONTENTS

  Every agreement of five pages or more MUST include an automatically
  generated Table of Contents. The TOC must:
    - Be inserted immediately after the title block (before Section 1)
    - Be generated using Insert > Table of Contents / Index feature
      so it updates dynamically with heading changes
    - Include all Heading 1 and Heading 2 entries
    - NOT be manually typed (a manually typed TOC violates standards)

2.2 PAGE NUMBERING

  All pages after the title page must display a page number.
    - Page numbers must appear in the document FOOTER
    - The page number must be centered in the footer
    - Use Insert > Header and Footer > Footer to add the footer,
      then insert the page number field via Insert > Field > Page Number
    - Do NOT manually type page numbers

2.3 DOCUMENT TITLE

  The document title (e.g., "COMMERCIAL LEASE AGREEMENT") must remain
  as a plain bold/centered paragraph — the "Title" paragraph style
  is optional but acceptable. The title should NOT be changed to a
  numbered section heading.

SECTION 3: ATTORNEY REVIEW CHECKLIST
═══════════════════════════════════════════════════════════════════

Before submitting for attorney review, the paralegal must confirm:
  [ ] All main section headings use "Heading 1" style (not direct bold)
  [ ] All subsection headings use "Heading 2" style (not direct bold)
  [ ] An automatically-generated TOC is present after the title
  [ ] Page numbers appear in the footer on all pages
  [ ] Document is saved in ODT format (.odt), not DOCX or PDF
  [ ] Final file name follows naming convention: [matter_description]_final.odt
