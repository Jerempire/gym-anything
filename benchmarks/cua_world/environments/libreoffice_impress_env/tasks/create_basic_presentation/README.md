# Create Basic Presentation Task

**Difficulty**: 🟢 Easy  
**Skills**: Slide creation, text entry, navigation  
**Duration**: 180 seconds  
**Steps**: ~50

## Objective

Create a new presentation with 5 slides, each containing a title and 2-3 bullet points about "Artificial Intelligence". This task tests fundamental presentation creation skills including slide management, text entry, and content organization.

## Task Description

The agent must:
1. Start with a blank LibreOffice Impress presentation
2. Create a total of 5 slides
3. Add a descriptive title to each slide
4. Add 2-3 bullet points to each slide with relevant content
5. Save the presentation

## Expected Structure

**Slide 1**: Introduction to AI
- Title: "What is Artificial Intelligence?"
- Bullets: Definition, key concepts, brief history

**Slide 2**: Types of AI
- Title: "Types of AI Systems"
- Bullets: Categories, examples, differences

**Slide 3**: Applications
- Title: "AI Applications"
- Bullets: Real-world uses, industry examples

**Slide 4**: Benefits
- Title: "Benefits of AI"
- Bullets: Advantages, improvements, efficiency gains

**Slide 5**: Future
- Title: "Future of AI"
- Bullets: Trends, predictions, emerging technologies

## Verification Criteria

1. ✅ **Slide Count**: Presentation has exactly 5 slides
2. ✅ **Titles Present**: At least 4/5 slides have non-empty titles
3. ✅ **Bullet Points**: At least 4/5 slides have 2+ bullet points
4. ✅ **Meaningful Content**: At least 4/5 slides have substantive content (not placeholders)

**Pass Threshold**: 75% (3/4 criteria must pass)

## Skills Tested

- Creating new slides (Ctrl+M or Insert menu)
- Entering slide titles
- Adding bullet points
- Navigating between slides
- Text editing and formatting
- File save operation

## Key Operations

- **New Slide**: Ctrl+M or Insert → New Slide
- **Navigate Slides**: Page Up/Down or slide pane
- **Edit Text**: Click text box, start typing
- **Save**: Ctrl+S
