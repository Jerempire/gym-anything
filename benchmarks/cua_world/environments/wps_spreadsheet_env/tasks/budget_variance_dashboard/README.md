# Budget Variance Dashboard

## Occupation
General and Operations Managers

## Industry
Administrative and Support and Waste Management and Remediation Services

## Difficulty
very_hard

## Description
Annual budget variance analysis across 8 cost centers with 28 expense line items over 12 months. Agent must create monthly variance calculations, YTD summaries with trend indicators, an executive summary using LARGE/INDEX-MATCH to identify worst/best performers, apply multi-level conditional formatting, and create two distinct charts.

## Data Source
Operational budget and actuals data modeled on real mid-market company departmental spending patterns, including realistic mid-year adjustments (rent increases, personnel raises, unplanned purchases).

## Features Exercised
- Cross-sheet variance formulas (Budget and Actuals references)
- Advanced lookup formulas (LARGE, SMALL, INDEX-MATCH for top/bottom analysis)
- Conditional formatting with icon sets (arrows) and color coding
- Clustered bar chart (budget vs actual by cost center)
- Line trend chart (monthly variance progression)
- YTD cumulative calculations with SUM ranges

## Verification Criteria (6 criteria, 100 points)
1. Monthly_Variance sheet with cross-sheet formulas (20 pts)
2. YTD_Summary with cumulative calculations and trend (20 pts)
3. Executive_Summary with LARGE/INDEX-MATCH analysis (20 pts)
4. Conditional formatting on multiple sheets (15 pts)
5. Clustered bar chart (12 pts)
6. Line trend chart (13 pts)

## Do-Nothing Score
0 - Starter workbook has only 2 data sheets; verifier checks for 3 new analysis sheets with formulas and charts.
