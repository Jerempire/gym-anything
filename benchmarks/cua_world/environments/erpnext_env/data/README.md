# ERPNext Demo Data

Real demo data sourced from the official ERPNext demo repository:
https://github.com/sahadnk72/erpnext-demo/tree/master/erpnext_demo/demo_docs

This data is designed for the "Wind Power LLC" company (a wind turbine manufacturer),
which is the company created by `erpnext.setup.utils.before_tests`.

## Data Files

| File | Source | Description |
|------|--------|-------------|
| `customers.csv` | Customer.csv | 19 real company names used as customers |
| `suppliers.csv` | Supplier.csv | 10 real company names used as suppliers |
| `items.csv` | Item.csv + Item_Price.csv | 17 real wind turbine parts and products with prices |
| `employees.csv` | Employee.csv | 15 real employee records with diverse international names |
