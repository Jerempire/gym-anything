# duplicate_merge

**Difficulty**: hard
**Timeout**: 720s | **Max steps**: 100

## Goal

Your Zotero library contains 10 neuroscience papers, but each paper was accidentally imported twice — resulting in 20 items total. Each original copy has a researcher note attached to it.

Merge all duplicates so that:
1. Exactly **one copy** of each paper remains (10 items total)
2. Each surviving item **retains its associated note**

Do not delete any papers outright — use Zotero's merge functionality to consolidate duplicates while preserving annotations.

## Success Criteria

- All 10 duplicate pairs merged (20 items → ~10 items)
- No paper titles appear more than once in the library
- Each of the 10 surviving papers still has at least one note attached
