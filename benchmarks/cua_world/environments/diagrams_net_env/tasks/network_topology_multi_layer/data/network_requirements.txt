=== DEVICE SPECIFICATIONS ===

CORE LAYER (2 devices):
  Core-SW-01: Cisco Nexus 9508, 40Gbps uplinks to both border routers
  Core-SW-02: Cisco Nexus 9508, 40Gbps uplinks to both border routers
  Core-to-Core VSS link: 40Gbps (virtual switching domain)
  Use Cisco multilayer switch shape from network library

DISTRIBUTION LAYER (4 devices):
  Dist-SW-01A: Cisco Catalyst 9500 (Floor 1 Left stack)
  Dist-SW-01B: Cisco Catalyst 9500 (Floor 1 Right stack)
  Dist-SW-02A: Cisco Catalyst 9500 (Floor 2 Left stack)
  Dist-SW-02B: Cisco Catalyst 9500 (Floor 2 Right stack)
  Each distribution switch connects to BOTH Core-SW-01 and Core-SW-02
  Inter-distribution redundancy: 10Gbps LAG between A/B pairs per floor
  Core-to-Dist links: 10Gbps each

ACCESS LAYER (8 devices):
  Access-SW-F1-01 through Access-SW-F1-04: Floor 1 (4 switches)
  Access-SW-F2-01 through Access-SW-F2-04: Floor 2 (4 switches)
  Each access switch connects to its floor's Dist A and Dist B switch
  Distribution-to-Access links: 1Gbps each (PoE+ capable)
  Use workgroup switch shape from Cisco network library

=== LINK LABELING ===
All edges must show: capacity / protocol (e.g., "40Gbps / OSPF")
WAN links: 1Gbps / eBGP (already present)
Core links: 40Gbps / OSPF Area 0
