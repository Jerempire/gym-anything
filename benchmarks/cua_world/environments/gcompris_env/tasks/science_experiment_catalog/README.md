# Science Experiment Catalog

## Task Overview

**Occupation**: Science Curriculum Developer / NGSS Alignment Specialist
**Difficulty**: Hard
**Timeout**: 600 seconds / 70 max steps

## Domain Context

Curriculum developers working with Next Generation Science Standards (NGSS) regularly audit educational software's science content for standards alignment. This involves personally running each science activity to understand the underlying concept being modeled, checking for scientific accuracy, and producing a formal curriculum alignment report that school districts use to make purchasing and adoption decisions.

## Goal

Produce a **NGSS Curriculum Alignment Report** at `~/Desktop/science_curriculum_report.txt` after:
1. Navigating to the Science/Experiment section
2. Opening and interacting with at least 3 experiment activities (covering physics, color/optics, and others)
3. Documenting each activity with its scientific concept, grade level, and NGSS recommendation

## Success Criteria

The report file `~/Desktop/science_curriculum_report.txt` must:
- Be created after the task started
- Be at least 450 bytes
- Mention at least 3 distinct science activities by name or concept
- Cover both physics-related and color/optics experiments

## Scoring (100 points)

| Criterion | Points |
|-----------|--------|
| Report file exists | 10 |
| Report created after task started (gate) | 10 |
| Report is ≥450 bytes | 10 |
| Physics/gravity/water experiment documented | 20 |
| Color/mixing/optics experiment documented | 20 |
| Other science activities documented | 15 |
| NGSS/grade-level/recommendation content | 15 |

Pass threshold: **60 points**

## GCompris Science/Experiment Activities (Reference)

Activities in the Science/Experiment sub-tab include:
- **Gravity**: Demonstrates gravitational force concepts
- **Watercycle**: Models the water cycle (evaporation, condensation, precipitation)
- **Mixing paint colors**: Demonstrates subtractive color mixing (CMY model)
- **Mixing light colors**: Demonstrates additive color mixing (RGB model)
- **Operate a canal lock**: Models hydraulic principles
- **Binary bulbs**: Demonstrates binary number representation (computer science)
- **Explore farm animals**: Life science / animal classification

## Notes

- The Science section is accessed via the pig/science icon in GCompris's category bar
- The "Experiment" sub-tab is the main focus (there are also History and Geography tabs)
- Each experiment should be run sufficiently to understand the scientific model it presents
