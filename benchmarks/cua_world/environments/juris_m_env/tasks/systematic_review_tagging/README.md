# systematic_review_tagging

## Overview
A law professor's Juris-M library contains 14 court cases for a systematic review on First Amendment (free speech) jurisprudence. The free-speech cases have inconsistent tags, and the library includes unrelated cases mixed in. The professor must organize, standardize tags, and annotate the library.

## Goal
1. Create 'Included Studies' collection with all 9 free speech cases
2. Create 'Excluded Studies' collection with all 5 unrelated cases
3. Standardize tags: all included cases should only have tag 'free-speech' (hyphenated, lowercase)
4. Add notes with holdings (>=40 words) to at least 4 included cases

## Scoring
- 'Included Studies' collection with >=6 correct free-speech cases: 25 pts
- 'Excluded Studies' collection with >=3 correct excluded cases: 15 pts
- Consistent 'free-speech' tag on >=6 free-speech cases: 20 pts
- Inconsistent tags removed from included items: 10 pts
- Notes with content >=40 words on >=4 included cases: 30 pts
- Pass threshold: 60 points
