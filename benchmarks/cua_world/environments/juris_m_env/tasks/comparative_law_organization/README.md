# comparative_law_organization

## Overview
A comparative law professor's Juris-M library has been bulk-imported from multiple sources. It contains 12 real court cases from three jurisdictions (United States, United Kingdom, Canada) all thrown into a flat list with no organization. Additionally, there are citation errors: three US cases are missing the court field, and two cases have the court incorrectly set to "Unknown Court".

## Goal
Organize this library and fix errors so it is ready for a comparative constitutional law course.

## Success Criteria
1. Create collection hierarchy: parent 'Comparative Constitutional Law' with subcollections 'US Cases', 'UK Cases', 'Canada Cases'
2. Assign all 12 cases to the correct subcollection by jurisdiction
3. Fix 3 missing court fields (set to 'United States Supreme Court')
4. Fix 2 wrong court fields (set to correct court names)
5. Tag all cases with their jurisdiction ('United States', 'United Kingdom', 'Canada')

## Scoring
- Parent collection + 3 subcollections: 25 pts
- US cases assigned correctly (>=4/5): 20 pts
- UK cases assigned correctly (>=3/4): 15 pts
- Canada cases assigned correctly (all 3): 10 pts
- Missing court fields fixed (>=2/3): 15 pts
- Wrong court fields fixed (>=1/2): 5 pts
- Jurisdiction tags (>=8 items): 10 pts
- Pass threshold: 60 points
